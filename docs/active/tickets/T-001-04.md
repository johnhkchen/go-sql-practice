## Context

Use Go's `embed` directive to bundle the Astro build output into the Go binary. Configure PocketBase to serve these static files on the root path so that hitting `/` in the browser renders the Astro frontend.

## Acceptance Criteria

- A Go file uses `//go:embed` to embed `frontend/dist/*`
- PocketBase serves the embedded files as the default frontend (root `/` path)
- Navigating to `http://localhost:8090/` in a browser shows the Astro index page
- PocketBase admin UI still works at `/_/`
- API routes (`/api/*`) still work and are not shadowed by the static file server
- Build instructions are documented in a Makefile or README section: build frontend first, then `go build`
