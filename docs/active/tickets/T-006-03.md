## Context

Astro page for the admin to control a sync session. Shows a slider bound to the 0-to-1 progress value and pushes updates to the server on change. Also displays the shareable viewer URL for easy copy/paste.

## Acceptance Criteria

- Admin page at `/sync/[id]/control` (Astro dynamic route)
- Reads `token` from the URL query string
- Displays a range slider (0 to 1, step 0.001) that sends `POST /api/sync/:id/progress` on input
- Shows the current progress value as a number next to the slider
- Shows the viewer URL (`/sync/:id`) with a copy-to-clipboard button
- Debounces or throttles slider updates (e.g., at most ~30 updates/sec) to avoid flooding the API
- Shows an error state if the token is invalid (403 from API)
- Uses an Astro client island (`client:load`) for the interactive slider
