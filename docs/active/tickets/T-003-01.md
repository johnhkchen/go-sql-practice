## Context

Implement `GET /api/links/search` with full-text search and optional tag filtering. This is the primary practice target for writing SQL queries against PocketBase's underlying SQLite database via the DAO layer.

## Acceptance Criteria

- `GET /api/links/search?q=golang` returns links whose title or description match the query
- `GET /api/links/search?tag=database` returns links tagged with "database"
- Both `q` and `tag` can be combined: `?q=intro&tag=golang`
- Results are paginated with `page` and `perPage` query params (defaults: page=1, perPage=20)
- Response format: `{"items": [...], "page": 1, "perPage": 20, "totalItems": 42}`
- Empty results return `{"items": [], ...}` with 200, not 404
- SQL injection is not possible (parameterized queries)
- Tag filter matches by slug, not by tag record ID
