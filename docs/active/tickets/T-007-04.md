## Context

The viewer page for a live presentation. Extends the basic sync viewer from S-006 with step awareness — viewers see which step they're on, step labels, and a step-based progress indicator instead of a raw progress bar. This is what the presenter shares with their audience.

## Acceptance Criteria

- Viewer page at `/watch/[id]` where `id` is the presentation ID (not the session ID)
  - Resolves the presentation's `active_session` to get the sync session, then subscribes
  - Shows "This presentation is not live" if `active_session` is null
- Displays:
  - Presentation name as page title
  - Current step: "Step 3 of 5 — [step label]" (computed from progress value and step_count)
  - Step progress indicator (dots/pills showing all steps, current one highlighted)
  - A progress bar showing fine position within the overall presentation
- Step computation: `current_step = round(progress * (step_count - 1))`
- Live updates via PocketBase realtime subscription on the sync_session record
- Smooth transitions between steps (CSS transition on progress bar, subtle animation on step change)
- Connection status indicator (connected / reconnecting / disconnected)
- Works on mobile — responsive layout, readable on phone screens during a call
- Clean, minimal design — this is what the audience sees, so it should look polished
