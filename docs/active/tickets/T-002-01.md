## Context

Create PocketBase collections for the link bookmarking app using Go migrations. PocketBase supports programmatic collection creation through its migration system, which is preferable to manual admin UI setup because it's reproducible and version-controlled.

## Acceptance Criteria

- Migration file(s) exist in a `migrations/` package
- **tags** collection: fields `name` (text, required), `slug` (text, required, unique)
- **links** collection: fields `url` (url, required), `title` (text, required), `description` (text), `tags` (relation to tags, multiple), `view_count` (number, default 0), `created_by` (relation to users)
- Migrations are registered in `main.go` and run on app startup
- Starting the app with a fresh database creates both collections
- Collections are visible in the PocketBase admin UI after startup
