## Context

Create the Astro frontend project inside the repository (e.g., `frontend/`). Configure it for static output so the build artifacts can be embedded into the Go binary. Set up a minimal index page as a smoke test.

## Acceptance Criteria

- `frontend/` directory contains a valid Astro project with `astro@5.17.3`
- `astro.config.mjs` sets `output: 'static'`
- `package.json` specifies `"engines": {"node": ">=24"}`
- `npm run build` (or equivalent) produces files in `frontend/dist/`
- Index page at `frontend/src/pages/index.astro` renders a basic "Link Bookmarks" heading
- `.gitignore` excludes `frontend/node_modules/` and `frontend/dist/`
