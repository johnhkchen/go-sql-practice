## Context

The frontend uses raw `EventSource` for PocketBase realtime subscriptions and manual `fetch()` for API calls. The PocketBase JS SDK (`pocketbase` npm package) provides a typed client with built-in realtime subscription management, auto-reconnect, and auth helpers. Installing it would simplify the sync/presentation viewer code and make it more robust.

## Acceptance Criteria

- `pocketbase` npm package installed in `frontend/package.json`
- Create a shared PocketBase client instance (e.g., `frontend/src/lib/pb.ts`)
- Refactor `/sync/[id]` (sync viewer) to use `pb.collection('sync_sessions').subscribe()`
- Refactor `/watch/[id]` (presentation viewer) to use SDK subscriptions
- Existing functionality unchanged — viewers still get realtime updates
- SDK handles reconnection automatically (remove manual reconnect logic)
