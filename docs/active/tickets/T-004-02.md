## Context

Build the home page that lists all links with their tags and provides a search interface. This is the primary UI and exercises Astro's client-side island architecture — the page shell is static, but search is interactive.

## Acceptance Criteria

- Home page (`/`) fetches links from PocketBase's built-in `GET /api/collections/links/records` on load
- Each link displays: title, URL (as clickable link), description snippet, tag pills, view count
- Search bar at the top sends queries to `GET /api/links/search` and updates the list
- Tag pills are clickable and navigate to `/tags/:slug`
- Client-side JavaScript handles search (Astro island with `client:load` or `client:visible`)
- Page works with JavaScript disabled (initial list renders, search just won't function)
