## Context

Build the link detail page that shows full information for a single link and increments the view counter. Exercises Astro's dynamic routes and client-side API calls.

## Acceptance Criteria

- Dynamic route at `/links/[id]` renders a single link's full details
- Page calls `POST /api/links/:id/view` on load to increment view count
- Displays: title, full URL, description, all tags (as pills linking to `/tags/:slug`), current view count
- View count updates visually after the increment call completes
- 404-style message if the link ID doesn't exist
