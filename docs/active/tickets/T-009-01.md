## Context

`routes/static.go` has an early `return` on line 36 that disables all static file serving. `registerStatic(e)` is commented out in `routes/routes.go`. The Astro frontend is embedded via `internal/frontend/embed.go` but never served. Hitting `/` returns nothing.

## What's Broken

- `static.go` line 36: early return bypasses all code
- `routes.go` line 19: `registerStatic(e)` commented out
- Multiple TODOs reference PocketBase v0.36.5 compatibility issues

## Acceptance Criteria

- `registerStatic(e)` is uncommented and working
- `static.go` serves the embedded frontend files on `/`
- PocketBase admin UI still works at `/_/`
- API routes (`/api/*`) are not shadowed by static file serving
- `go build` compiles without errors
- Navigating to `http://localhost:8090/` shows the Astro index page
