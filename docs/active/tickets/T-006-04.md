## Context

Astro page for viewers to watch a sync session's progress in realtime. Subscribes to PocketBase's realtime API (SSE) for the session record and renders a progress bar that updates live as the admin moves the slider.

## Acceptance Criteria

- Viewer page at `/sync/[id]` (Astro dynamic route)
- On load, fetches the current session record to get initial progress value
- Subscribes to PocketBase realtime updates for the specific `sync_sessions` record
- Renders an HTML `<progress>` element (or styled equivalent) that reflects the current 0-to-1 value
- Displays the numeric progress value (e.g., "42.0%")
- Updates are visually smooth (CSS transition on the progress bar width)
- Shows a "Session not found" message if the session ID doesn't exist
- Shows a connection status indicator (connected / reconnecting / disconnected)
- Uses PocketBase JS SDK for the realtime subscription (`pb.collection('sync_sessions').subscribe(id, callback)`)
- Works on mobile browsers
