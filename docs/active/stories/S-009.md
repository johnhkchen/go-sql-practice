## Cleanup and Fixes

Post-implementation sweep to fix broken, incomplete, or insecure code left behind by the initial ticket pass. These are things that were marked done but aren't actually working correctly.

## Issues Found

1. Static file serving is completely disabled — the embedded Astro frontend isn't served
2. SQL injection vulnerability in `links_search_simple.go` string interpolation
3. `links_view.go` returns incomplete response (can't access DAO in PB v0.36.5)
4. Integration tests use mock HTTP responses instead of real endpoint tests
5. `StatsSummary.astro` hardcodes API URL to `http://127.0.0.1:8094`
6. PocketBase JS SDK not installed — manual EventSource for realtime
7. `PathValue()` may not work with PocketBase v0.36.5's router

## Goal

Every endpoint works, every page connects to the right API, tests actually test things, and there are no security vulnerabilities.
