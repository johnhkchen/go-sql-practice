## Custom Go API Routes

Build custom Go API endpoints that go beyond PocketBase's auto-generated CRUD. These exercise SQL queries, request validation, and business logic in Go.

## Goal

Implement the following custom endpoints:
- `GET /api/links/search?q=&tag=` — full-text search with optional tag filter, returns paginated results
- `POST /api/links/:id/view` — atomically increment view_count, return updated count
- `GET /api/stats` — aggregate statistics (total links, top tags, most viewed)

## Why

PocketBase's built-in API handles basic CRUD. These custom routes practice writing raw SQL queries, using PocketBase's DAO layer, handling query parameters, input validation, and returning structured JSON responses from Go.
