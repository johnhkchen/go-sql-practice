## Tech Debt & Code Quality Cleanup

Three audit agents (Go backend, Astro frontend, project infrastructure) found ~70 issues across the codebase. S-009 covers the critical functional bugs (static serving, SQL injection, pathvalue routing, test mocks, hardcoded URLs, PB JS SDK). This story covers the remaining tech debt, code quality, and low-hanging optimizations that S-009 doesn't address.

## Scope

1. Dead code and duplicate removal
2. Shared Go utility extraction and consistency fixes
3. Frontend shared types and API client
4. Frontend HTML/CSS/accessibility cleanup
5. Project infrastructure (CI, Makefile, .gitignore)
6. Inline script extraction from Astro components

## Goal

Clean, consistent codebase with no dead code, shared utilities instead of copy-paste, proper CI, and a maintainable frontend architecture.
