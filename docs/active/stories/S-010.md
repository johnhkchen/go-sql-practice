## Tech Debt & Code Quality Cleanup

S-009 covers the critical functional bugs (static serving, SQL injection, pathvalue routing, test mocks, hardcoded URLs, PB JS SDK). This story covers the remaining tech debt, code quality, and low-hanging optimizations that S-009 doesn't address.

7. Fix build failures from cleanup pass
